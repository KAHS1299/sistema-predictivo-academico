Linear Regression Explanation

What is Linear Regression?

Linear regression is a supervised Machine Learning algorithm used to predict a continuous value based on one or more input variables.

How does it work?

It finds the best straight line that fits the data. This line represents the relationship between the independent variables (inputs) and the dependent variable (output).

Mathematical Model:

y = β₀ + β₁x₁ + β₂x₂ + ... + βₙxₙ

  • y: predicted value
  • x: input variables
  • β: coefficients learned by the model

Goal:

Minimize the error between predicted values and real values.

Application in This Project

Problem:

Predict house prices using data from the Boston Housing dataset.

Input Variables:

  • RM - Number of rooms
  • LSTAT - Lower status population (%)
  • CRIM - Crime rate

Output:

MEDV - Median house value

How the model works here:

The model analyzes how each variable affects the house price and combines them to generate a prediction.

Example:

Houses with more rooms (RM) tend to have higher prices, while higher crime rates (CRIM) reduce property value.
