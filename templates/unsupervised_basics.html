Unsupervised Machine Learning

Unsupervised learning identifies hidden patterns and structures in data without labeled outputs. Explore the core concepts and mechanics of the K-Means algorithm below.

Clustering

Grouping data points where elements in the same group are more similar to each other than to those in other groups.

K-Means Algorithm

Partitions data into K groups defined by a centroid. Points are assigned based on proximity.

Centroids

The center of a cluster, calculated as the average position of all points within it.

Distance Metrics

Euclidean distance is the most common metric used to measure how close or far data points are.

Iterative Process

The algorithm repeatedly assigns points and recalculates centroids until clusters stabilize.

Real-World Applications

Used in customer segmentation, anomaly detection, and pattern recognition.
