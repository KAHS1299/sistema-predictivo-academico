Unsupervised Machine Learning

Unsupervised learning is a type of machine learning that identifies patterns, structures, and relationships in data without the use of labeled outputs. The model explores the data and groups or organizes it based on similarities.

Clustering

Clustering is the process of grouping data points into clusters where elements in the same group are more similar to each other than to those in other groups.

K-Means Algorithm

K-Means is a clustering algorithm that partitions data into K groups. Each group is defined by a centroid, and data points are assigned based on proximity.

Centroids

A centroid represents the center of a cluster. It is calculated as the average position of all data points within a cluster.

Distance Metrics

Distance measures how close or far data points are. The most common metric used in K-Means is the Euclidean distance.

Iterative Process

The algorithm repeatedly assigns points to clusters and recalculates centroids until the clusters stabilize and no longer change significantly.

Variance and Convergence

K-Means minimizes the variance within each cluster. Convergence is reached when centroids stop moving or the change becomes negligible.

Real-World Applications

Unsupervised learning is used in customer segmentation, anomaly detection, recommendation systems, image compression, and pattern recognition.
