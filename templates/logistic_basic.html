Logistic Regression - Basic Concepts

What is Logistic Regression?

Logistic regression is a supervised Machine Learning algorithm used for classification problems, especially binary classification.

Regression vs Classification:

  • Regression → predicts continuous values (e.g., price)
  • Classification → predicts categories (e.g., pass/fail)

Binary Classification:

It predicts one of two possible outcomes (0 or 1).

Target Variable:

The output class label (e.g., 0 = Fail, 1 = Pass)

Features:

Input variables used to make predictions (hours, attendance, etc.)

Key Concepts

Probability Estimation:

Logistic regression predicts probabilities between 0 and 1.

Decision Boundary:

A threshold that separates classes (usually 0.5).

Classification Threshold:

If probability > 0.5 → class = 1 If probability ≤ 0.5 → class = 0

Real-world Example:

Predict if a student will pass or fail based on study habits.
