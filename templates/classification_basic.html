What is Classification?

Unlike Linear Regression (which predicts continuous numerical values, like a house price), Classification aims to assign a label or category to the input data.

It is the process of learning a function that maps an input vector of variables into one of several discrete classes.

Key Examples:

  • Spam Detection: Is it Junk or Safe?
  • Medical Diagnosis: Malignant or Benign?
  • Academic Performance: Will the student Pass or Fail?

Our Model: AdaBoost

Adaptive Boosting Algorithm

1. Weak Learners

AdaBoost uses multiple "Decision Stumps" (mini-models) that are simple but work together.

2. Sequential Learning

Each new model focuses specifically on correcting the errors made by the previous model.

3. Weighted Voting

Finally, all mini-models vote, and the majority decision determines the final classification.
